How does this work?

Just input your relation schema and any given functional dependencies and hit submit! ClozureDB+ will calculate and display as much useful information about your schema as it can.

What information will ClozureDB+ give me?

ClozureDB+ is in a heavy development phase. Currently, I am working on parsing and validating input. Once that is complete, the first feature to be implemented will be the calculation of attribute closures and keys/superkeys. Check out the Github and reach out if you are interested in contributing!

Formatting Info Relation format:
TITLE(A,B,C,D,...,Z)
FD format:
A->B ; C->A,B ; A,D->C

Relation title is optional.
Input is case insensitive and spaces are ignored.
Input must not contain numbers.
